    NodeJS password vault - desktop and mobile application.

    Project maintained by perry-mitchell

    Filling a gap

    Password managers have been around a long time, though they're thoroughly under-utilised. Having a reliable product that functions remotely and on every operating system is still a dream - at least for those wanting a free alternative.

    Buttercup is an open-source, cross-platform password management application built for a couple of very simple reasons:

    • There is no free manager that works well across every operating system
    • There is no free manager that has a unified UX and UI across every operating system

    WIP

    Buttercup is a work-in-progress. Expect a working beta before EOY 2015.
